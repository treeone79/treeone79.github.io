  1. 出力

    • C言語では、printfという関数を使って出力を行います。

      • printfのfは、formatの略
      • 使い方は、printf("hogehoge")のように、("") の中に書いた文字が出力される。

    • 実際に、書いてみよう。

      #include <stdio.h>
      int main(){
         printf("Hello, World\n");
         return 0;
      }
    • 書いたら、実行してみましょう。
      • 正しく書けていれば、Hello, World と表示されるはずです。
      • printfの中の\nは、改行を表す。
    • 実行結果が、正しかったら、提出してみよう！！
      • 問題ページにアクセスして、右上の提出フォームから、提出してみよう！！
    • 正解すると、 AC と表示されます。
      • ACはAcceptの略で、プログラムが正しい結果を返したことを表す。
      • 他にも、以下のような状態がある。
        略称 名称 意味
        AC Accept 正解、正しい答えを返した
        WA Wrong Answer 誤答、間違った結果を返した
        TLE Time Limit Exceeded 制限時間内にプログラムが終了しなかった
        MLE Memory Limit Exceeded メモリ制限を超えてメモリを使用した
        RE Runtime Error プログラム実行中にエラーが発生した

  2. 変数

    • プログラム中で、数字や文字などのデータを扱うためには、変数 というものを使う。
      • 変数とは、データを保存するための箱のようなもの。
    • 変数を使うためには、まず、宣言 を行う必要がある。
      • 型名　変数名;の順で宣言を行う。
      • 型は、扱うデータの種類を表す。
        型名種類サイズ範囲
        int整数型32bit-2^31~2^31-1
        long long整数型64bit-2^63~2^63-1
        double浮動小数点数型64bitおおよそ-10^308~10^308で
        15桁の精度
        char文字型8bit-2^7~2^7-1
      • 例えば、int型の変数numを宣伝する場合は、int num;のように書く。
    • 変数には、=を使って、値を代入することができる。
      • プログラム中では、= は等号ではなく、代入 として用いる。
      • numに10を代入するコードは、num=10;のように書く。
      • 文字を代入するときは、文字を 'a' のように、シングルクウォーテーションで囲む。
      • 文字自体も一つの数字である（ ASCIIコード等）。
    • 変数は、四則演算を行うことができる。
      • 例えば、num1とnum2をたす時には、num1+num2 のように書く。
        演算記号
        足し算 +
        引き算 -
        掛け算 *
        割り算（商） /
        割り算（余り） %
    • 変数の値を出力するには、printfの フォーマット指定子 を用いる。
      • 例えば、int型の変数numを出力するときは、printf("%d", num)のように書く。
      • この%dの部分が、フォーマット指定子と呼ばれていて、,の後の変数を出力する。
        型名フォーマット指定子
        int%d
        long long%ld
        double%lf
        char%c
    • 例えば、100と17について、四則演算の結果を表示するプログラムは、以下のように書く。
      #include<stdio.h>
      int main(){
         int num1 = 100, num2 = 17;
         printf("%d+%d=%d\n", num1, num2, num1+num2);
         printf("%d-%d=%d\n", num1, num2, num1-num2);
         printf("%d*%d=%d\n", num1, num2, num1*num2);
         printf("%d/%d=%d...%d\n", num1, num2, num1/num2, num1%num2);
         return 0;
      }

  3. 入力

    • 次の 問題 解いてみよう！
      • 標準入力から、整数A, Bが与えられる。
      • A+Bを１行で出力せよ。

        • 変数名の前につける & を忘れないようにする。

    • コード例
      #include<stdio.h>
      int main(){
         int num1, num2;
         scanf("%d %d", &num1, &num2);
         printf("%d\n", num1+num2);
         return 0;
      }
    • 書けたら、提出してみよう！！
    • ACできたら、次の 問題も解いてみよう。
      • 標準入力から小数A, Bが与えられる。
      • A+Bを1行で出力せよ。

  4. 条件分岐

    • 次の 問題 を解いてみよう！
      • 標準入力から、西暦が与えられる。
      • 与えられた年が、うるう年なら leap year を、そうでないなら no leap year を出力せよ。
      • なお、うるう年の条件は、次の方法で判定できる。
        • 4の倍数である
        • 100の倍数でない
        • ただし、400の倍数は、うるう年である
    • この問題を解くためには、うるう年かどうかで、条件分岐を行う必要がある。
    • 条件分岐には、if文を使う。
      • 使い方は、if(条件式)で条件式が満たされた時に、次の１文が実行される。
      • {}で囲むことによって、複数の文を1文として、扱うことができる。
      • elseを使うことによって、条件式が成り立たなかった時に実行する文を記述できる。
      • 条件式 に使える記号は、以下のものがある。
        記号意味
        == 等号　左辺と右辺が等しい時に成り立つ
        != 左辺と右辺が異なる時に成り立つ
        < 左辺が右辺よりも小さい時に成り立つ
        > 左辺が右辺よりも大きい時に成り立つ
        <= 左辺が右辺以下の時に成り立つ
        >= 左辺が右辺以上の時に成り立つ
    • 例えば、numが0の時に、ZERO、それ以外の時に、NO ZERO と表示するコードは以下のようにかける。
      if(num == 0) printf("ZERO\n");
      else printf("NO ZERO\n");
    • 上記のコードは、{}を使って、以下のようにもかける。
      if(num == 0) {
         printf("ZERO\n");
      }
      else {
         printf("NO ZERO\n");
      }
    • また、複数の条件分岐を行いたいときは、以下のようにつなげて書くことができる。
      if(num > 0) printf("positive\n");
      else if(num < 0) printf("negative\n");
      else printf("zero\n");
    • 条件分岐の仕方がわかったところで、上記の問題を解いてみよう！！
    • AがBの倍数かどうかは、AをBで割った余りが0かどうかで判定できる。
    • コード例
      #include<stdio.h>
      int main(){
         int year;
         scanf("%d", &year);
         if(year%400 == 0) printf("leap year\n");
         else if(year%100 == 0) printf("no leap yaer\n");
         else if(yaer%4 == 0) printf("leap year\n");
         else printf("no leap year\n");
         return 0;
      }
    • 書けたら、提出してみよう！！

  5. 繰り返し(定数回)

    • 次の 問題 を解いてみよう！
      • 標準入力から整数Nが与えられる。
      • 1 <= i <= N の各iについて、
        • iが3の倍数なら、Fizzを、
        • iが5の倍数なら、Buzzを、
        • iが15の倍数なら、FizzBuzzを、
        • そうでないなら、iを
        • i行目に出力せよ。
    • 1<=i<=Nについて、繰り返しの処理ができれば、条件分岐を用いて解けそう！！
    • C言語では、for文 を使うことで、繰り返しができる。
      • 使い方は、for(初期化式; 条件式; 更新式)で、１回だけ呼ばれる初期化式、毎回呼ばれる条件式、更新式がある。
      • まず、for文に入ると、プログラムは、1回だけ 初期化式を行う。
      • その後、以下の動作を繰り返す。
        • 条件式が満たされていないなら、繰り返しを終了する。
        • for文の直後の1文を実行する。
        • 更新式を実行する
      • if文と同じように、{}で囲むことで、複数の文を、1文にすることができる。
      • 更新式のあとだけ;がないので注意する。
    • 例えば、1からNまでの数を出力するコードは次のようにかける。
      for(i = 1; i <= N; i++) printf("%d\n", i);
    • 繰り返しの方法がわかったところで、上の問題を、解いてみよう！
    • コード例
      #include<stdio.h>
      int main(){
         int N, i;
         scanf("%d", &N);
         for(i = 1; i <= N; i++) {
             if(i % 15 == 0) printf("FizzBuzz\n");
             else if(i % 3 == 0) printf("Fizz\n");
             else if(i % 5 == 0) printf("Buzz\n");
             else printf("%d\n", i);
         }
         return 0;
      }
    • 書けたら、提出してみよう！！

  6. 繰り返し（不定回）

    • 次の 問題 を解いてみよう！
      • 標準入力から、整数Aが与えられる。
      • 整数Aの桁に含まれる3の個数を出力せよ。
    • C言語では、for文以外に、while文を使って繰り返しを行うことができる。
    • 使い方は、while(条件式)で条件式が満たされている間、直後の１文が繰り返される。
      • if文やfor文と同じように、{}で囲むことによって、複数の文を1文として扱える。
    • 例えば、num1をnum2以上の最小の2冪にしたい場合は、次のようにかける。
      num1 = 1;
      while(num1 < num2) num1 *= 2;
    • 繰り返しの方法がわかったところで、上の問題を解いてみよう！
      • この問題では、下１桁が３かどうか見たあとに、10で割っていく操作を繰り返せば良い。
    • コード例
      #include<stdio.h>
      int main(){
         int A, ans = 0;
         scanf("%d", &A);
         while(A > 0) {
             if(A%10 == 3) ans++;
             A /= 10;
         }
         printf("%d\n", ans);
         return 0;
      }
    • 書けたら、提出してみよう！

  7. 配列

    • 次の 問題 を解いてみよう！
      • 標準入力から、整数Nと長さNの数列が与えられる (N <= 100)。
      • 数列を反転させたものを出力せよ。
    • この問題を特には、数列を一度変数に保存する必要がある。
      • int a, b, c, d, ...のように、最大で100個もの変数を作るのは、大変！！
    • C言語には、配列というものがあるので、それを使おう！！
      • 配列とは、複数の変数を連続するメモリに、格納したもの。
    • 使い方は、int num[100]のように、型名　変数名[サイズ]で宣言し、アクセスするときは、num[0]のように、変数名[インデックス]でアクセスできる。
      • インデックスは、0 で始まり、サイズ-1 で終わるので、注意する！！
      • 上記の例だと、num[0]~num[99]までは、アクセスできるが、num[100]は配列外参照と言ってエラーが起きる！！
    • 例えば、サイズ10の配列numを用意し、0~9までを代入するコードは、次のようになる。
      int num[10], i;
      for(i = 0; i < 10; i++) num[i] = i;
    • このように、配列とfor文の組み合わせはよく使うので、覚えておくと良い。
    • 配列の使い方がわかったところで、上の問題を解いてみよう！！
    • コード例
      #include<stdio.h>
      int main(){
         int N, a[110], i;
         scanf("%d", &N);
         for(i = 0; i < N; i++) scanf("%d", &a[i]);
         for(i = N-1; i >= 0; i--) printf("%d\n", a[i]);
         return 0;
      }
    • 書けたら、提出してみよう！！

  8. 文字列

    • 次の 問題 を解いてみよう！
      • 文字列Sが与えられる。
      • Sが回文かどうか判定せよ。
    • 文字列は、文字の配列で表される。
    • 文字列も、他の変数と同じように、入出力を行うことができる。
      • フォーマット指定子は%sを使う。
      • 入力の際は、他の変数と異なり、変数名の前に、&を つけない ので注意する。
        char s[110];
        scanf("%s", s);
        printf("%s\n", s);
    • C言語には、文字列を扱うための関数がたくさん用意されている。
      • 使用するためには、<string.h>をincludeする必要がある。
      • よく使うものを、以下に挙げる。
        関数名引数動作
        strcpychar* a, char* baにbをコピーする
        strcatchar* a, char* baの末尾にbを結合する
        strcmpchar* a, char* baとbを比較して、a < bなら負, a == bなら0, a > bなら正の値が返る
        strlenchar* a文字列aの長さを返す
    • 回文かどうか判定するには、文字列の長さがわかれば良さそう！！
    • コード例
      #include<stdio.h>
      #include<string.h>
      int main(){
         char s[110];
         int len, i, isKaibun = 1;
         scanf("%s", s);
         len = strlen(s);
         for(int i = 0; i <= len / 2; i++) {
             if(s[i] != s[len-i-1]) isKaibun = 0;
         }
         if(isKaibun) printf("Kaibun\n");
         else printf("no Kaibun\n");
         return 0;
      }
    • 書けたら、提出してみよう！！
    • ACできたら、次の 問題 を解いてみよう！
      • 大文字と小文字からなる文字列Sが与えられる。
      • 各文字について、大文字と小文字を反転したものを出力せよ。
    • この問題を解くためには、まず、文字cが大文字か小文字かを判定する必要がある。
    • プログラム内では、文字は数字として扱われており、'A'~'Z', 'a'~'z'はそれぞれ、連続しているので、if文を使って判定することができる。
      • 具体的な数字を知りたい場合は、ASCIIコードを参照。
      • コード例
        if(c>='A' && c <= 'Z') printf("%c is Oomoji\n", c);
        else if(c >= 'a' && c <= 'z') printf("%c is komoji\n", c);
        else printf("%c is not Alphabet\n", c);
    • また、文字cが何番目のアルファベットであるかは、大文字なら'A'との差を、小文字なら'a'との差をとることで、知ることができる。
      • コード例
        int n;
        if(c >= 'A' && c <= 'Z') n = c - 'A';
        else if(c >= 'a' && c <= 'z') n = c - 'a';
        printf("%c is %dth Alphabet\n", c, n);
    • よって、上の問題を解くコードは、以下のように書ける。
      • コード例
        #include<stdio.h>
        #include<string.h>
        int main(){
           char s[110];
           int len, i;
           scanf("%s", s);
           len = strlen(s);
           for(i = 0; i < len; i++) {
               if(s[i] >= 'A' && s[i] <= 'Z') s[i] += 'a' - 'A';
               else if(s[i] >= 'a' && s[i] <= 'z') s[i] += 'A' - 'a';
           }
           printf("%s\n", s);
           return 0;
        }
    • 書けたら、提出してみよう！！

  9. 関数
    • 次の 問題 を解いてみよう！
      • 整数Nが与えられる。
      • N!を出力せよ。
    • N! = 1 * 2 * ... * n -1 * n より、1からnまでfor文を回せば、答えが求まりそう。
    • コード例
      #include<stdio.h>
      int main(){
         int N, ans = 1, i;
         scanf("%d", &N);
         for(i = 1; i <= N; i++) ans = ans * i;
         printf("%d\n", ans);
         return 0;
      }
    • この処理は、関数を使っても実現できる。
    • 関数は、返り値の型　関数名(引数) {関数の中身; retrun 返り値;}の形で定義する。
    • 関数の呼び出しは、関数名(引数)で行う。
    • コード例
      #include<stdio.h>
      int func(int n) {
         if(n <= 1) return 1;
         return n * func(n - 1);
      }
      int main(){
         int N;
         scanf("%d", &N);
         printf("%d\n", func(N));
         return 0;
      }
